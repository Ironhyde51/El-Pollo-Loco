Class: Level

Level(enemies, clouds, backgroundObjects, coins, bottles, bottleBoxes)

Stores all objects that belong to one level.

Constructor

new Level(enemies, clouds, backgroundObjects, coins, bottles, bottleBoxes)

Creates a level with enemies, collectibles and background objects.
Parameters:
Name Type Description
enemies Array.<MovableObject> Enemies and endboss of the level.
clouds Array.<Clouds> Cloud objects shown in the level.
backgroundObjects Array.<BackgroundObject> Background layers of the level.
coins Array.<Coin> Collectible coins.
bottles Array.<Bottle> Collectible bottles.
bottleBoxes Array.<BottleBox> Bottle boxes that refill bottles.
