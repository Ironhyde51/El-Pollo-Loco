# 🌶️ El Pollo Loco

Willkommen bei **El Pollo Loco**!

Dieses Projekt ist ein 2D-Jump-and-Run-Spiel, das mit **HTML**, **CSS** und **JavaScript Canvas** umgesetzt wurde.

Der Spieler steuert Pepe durch eine Wüstenwelt, sammelt Münzen und Salsa-Flaschen, besiegt Gegner und kämpft am Ende gegen den Endboss.

## 💻 Tech Stack

Dieses Spiel wurde mit einem Vanilla-JS-Ansatz umgesetzt:

* **HTML5** - Grundstruktur der Anwendung
* **CSS3** - Layout, Responsive Design und UI-Elemente
* **JavaScript (ES6+)** - Spiellogik, Kollisionen, Animationen und Steuerung
* **Canvas API** - Darstellung der Spielwelt und Spielfiguren
* **JSDoc** - Dokumentation der Klassen und Funktionen

---

## ✨ Features

- 🎮 Spielbare 2D-Welt mit Kamera-Bewegung
- 🧍 Steuerbarer Charakter Pepe
- 🐔 Gegner und kleine Gegner
- 👑 Endboss mit Statusbar und Animationen
- 🪙 Münzen sammeln mit Statusbar
- 🌶️ Flaschen sammeln und werfen
- 📦 Bottle-Boxen als Nachfüll-Items
- ❤️ Health-System für Pepe
- 🔊 Hintergrundmusik und Soundeffekte
- 🖥️ Startscreen, Game-Over-Screen und Winning-Screen
- 📱 Touch-Steuerung für mobile Geräte
- 🔇 Mute-Funktion
- 🧾 Impressum und Info-Modal
- 📚 JSDoc-Dokumentation

## 🎮 Controls

| Taste | Aktion |
|------|--------|
| Pfeil Rechts | Nach rechts laufen |
| Pfeil Links | Nach links laufen |
| Leertaste | Springen |
| D | Flasche werfen |
| Sound-Button | Ton ein-/ausschalten |
| Fullscreen-Button | Vollbild aktivieren |
